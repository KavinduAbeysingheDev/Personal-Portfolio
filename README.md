Personal Portfolio Website

A modern, responsive personal portfolio website designed to showcase projects, technical skills, and professional experience. Built with a dark theme, smooth animations, and an interactive user experience.

Features

• Responsive Design
Optimized for desktop, tablet, and mobile devices. 
• Interactive UI
Smooth hover effects, glowing elements, and parallax-style interactions. 
• Smooth Navigation
Sticky navbar with a mobile-friendly hamburger menu and smooth scrolling. 
• Animated Statistics
Dynamic number counting animation in the About section. 
• Scroll Animations
Elements animate into view using the Intersection Observer API. 
• Contact Form
Simple mailto-based contact functionality for direct communication. 

Technologies Used

• HTML5 – Semantic structure 
• CSS3 – Flexbox, Grid, animations, and custom variables 
• JavaScript (Vanilla) – DOM manipulation and interactivity 

External Libraries

• Google Fonts (Poppins, Space Mono) 
• Font Awesome (Icons)
